Tentang Saya

Halo, saya Adhyaksa Fitrah. Saya adalah seorang pelajar yang bersemangat di bidang Pengembangan Perangkat Lunak dan Gim. Saya percaya bahwa setiap masalah adalah tantangan untuk belajar hal baru.


Portofolio ini adalah tempat saya mendokumentasikan proyek-proyek yang saya buat, baik dari tugas sekolah maupun proyek pribadi. Saya antusias untuk terus belajar dan berkontribusi pada dunia teknologi di masa depan.

Projek Saya

Website Profil Sekolah

Sebuah proyek tugas sekolah untuk membuat website profil sekolah yang informatif dan menarik. Dibuat menggunakan HTML dan CSS untuk struktur dan desainnya.

Halaman Kontak Sederhana

Halaman web sederhana yang menampilkan informasi kontak. Proyek ini berfokus pada pembuatan formulir dasar dan penataan elemen menggunakan HTML dan CSS.

Landing Page Produk

Sebuah landing page sederhana yang dirancang untuk memperkenalkan sebuah produk fiktif. Proyek ini berfokus pada desain responsif agar tampilan tetap baik di berbagai perangkat, menggunakan HTML5 dan CSS3.
